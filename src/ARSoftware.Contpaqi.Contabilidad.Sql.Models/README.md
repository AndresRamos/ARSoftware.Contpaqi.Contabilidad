# AR SOFTWARE CONTPAQI CONTABILIDAD SQL MODELS

Este proyecto contiene las clases mapeadas por Entity Framework que representan las tablas de las bases de datos de CONTPAQi Contabilidad de Generales y Empresas.

## Instalación

Puedes instalar el proyecto utilizando [NuGet](https://www.nuget.org/packages/ARSoftware.Contpaqi.Contabilidad.Sql.Models):

```
dotnet add package ARSoftware.Contpaqi.Contabilidad.Sql.Models
```
